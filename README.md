# Word-Tpl Word模板引擎
如果只是在word中替换变量，那是我们没用充分利用起word的功能，word-tpl实现了在word中使用我们常用的模板引擎来对word进行内容控制，比如if判断，for循环等，让生成报告更加方便简单。word-tpl中word操作采用POI进行编辑，而不是采用xml生成方式生成的，这样生成的word更加标准。
# 功能说明
* [x] 变量替换
* [x] 条件判断
* [ ] 段落内的for循环
* [ ] 段落间的for循环
* [ ] 表格间的for循环
